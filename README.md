# AI-Assisted Market Entry Diagnosis

A public, anonymized portfolio case page showing how structured problem solving and AI-supported workflow design were used to diagnose market-entry conversion bottlenecks from a China offline festival activation.

## What this is

This repository contains the public web version of a selected work sample:

- `index.html` — the portfolio case page
- `styles.css` — page styling
- `assets/market-entry-diagnosis.pdf` — redacted public PDF work sample

## What the case demonstrates

- Market-entry diagnosis
- Evidence synthesis from logged transactions and product-level sales records
- Consumer insight and journey diagnosis
- AI-assisted problem framing
- Human-in-the-loop reasoning control
- Decision-ready recommendation design

## Privacy note

This public version uses anonymized and summarized content only. Exact financial figures, raw transaction-level data, internal reports, and confidential brand documents are not included.
